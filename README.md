**Resume-Builder**
ResumeForge is a modern and responsive web-based resume builder that helps users create professional resumes quickly and easily. It provides a clean interface with real-time preview, allowing users to see changes instantly while editing.

**Features**
* Live Resume Preview
* Personal Information Section
* Work Experience, Education & Projects
* Skills, Certifications & Achievements
* Languages & Extra Sections
* Photo Upload Option
* Multiple Resume Templates
* Zoom Controls for Preview
* Print & Download as PDF
* Tab-based Navigation (Info, Experience, Extras, Design)

**Technologies Used**
* HTML
* CSS
* JavaScript
